This project is a small web application built with Flask that allows users to register and log in.

The frontend uses Jinja templates to render HTML pages dynamically, while the backend uses SQLAlchemy to connect to a PostgreSQL database for storing user information securely.

It includes basic form validation, ensures unique usernames and emails, and demonstrates how a full-stack web application handles user authentication, database integration, and templated frontend pages.
